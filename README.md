# Exercise Share App

運動の記録を投稿できるWebアプリ

## 機能

* プロフィール画像の変更
* ユーザー名の変更
* マイエクササイズの追加と削除
* マイエクササイズから選択して新規投稿
* コメントをつけて新規投稿
* nice機能
* ユーザーの投稿数の表示
* ユーザーのnice数の表示

## 使用技術

* React
* React hooks
* StyledComponents
* firebase
* Material-UI
